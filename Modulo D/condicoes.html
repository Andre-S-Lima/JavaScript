Condições

Até agora aprendemos a mexer com variaveis e como trabalhar com elas, mas tudo de uma forma sequencial. Apenas seguia na ordem na qual voce programava, por exemplo, você pegava a variavel n1, depois pegava a variavel n2 e por fim voce somava elas, você nao podia pular para a n2 e depois a n1 por exemplo. Isso é uma forma sequencial na programação. Agora iremos aprender a programar de forma condicional, ou seja, com condições.

Tipos de condições

Existem varios tipos de condições, mas vamos nos focar nessas abaixo. Que seriam as condições simples e condições if/else.

Condições Simples

Vamos utilizar o exemplo de dois valores, cada um dentro de uma variavel n1 e n2, e então iremos somar elas, só que você quer saber se ela é verdadeira. Utilizarems então a condição SE, ou if, para o programa verificar se é verdadeiro essa afirmação, se for verdadeira ele irá apontar, se não for, o programa rodará normalmente. Abaixo temos um exemplo da estrutura simples: 

                            n1 = valor
                            n2= valor
                            soma = n1 + n2
                                |
                                |
                                |
                        soma é maior que 5?
                                |
                        |-------+-------|
                        |               |
                    Sim (true)

Condição Composta

Imagina que temos então as duas variaveis como no exemplo acima e somamos elas, só que você precisa saber se esse numero é maior que 5 só que diferente do exemplo acima, se for verdadeiro ele terá alguma ação, e se for falso terá outra ação. Você ira utilizar a estrutura do if/else para verificar se é true ou false, true se for maior que 5 ou false se não for maior que 5. Abaixo temos um exemplo de como ficaria:

                        n1 = valor
                        n2= valor
                        soma = n1 + n2
                            |
                            |
                            |
                    soma é maior que 5?
                            |
                    |-------+-------|
                    |               |
                Sim (true)       Não(false)

Essa estrutura é a estrutura de condição IF, basicamente uma bifurcação na execução do seu programa, ela checa SE(if) o valor é verdadeiro, ou se é falso(else).

Condições Aninhandas

As condições aninhandas seguem o mesmo padrão da condição composta, utilizando o IF/ELSE, a diferença é que para ela funcionar a condição tem que ser falsa e ao ser falsa, você utiliza outra condição dentro do ELSE. Ela se chama ELSEIF, ou ELIF.

Switch Case

Até agora vemos como a estrutura IF/ELSE funciona, estrutura essa que é uma estrutura de condição. Agora iremos ver a estrutura de condição Switch. Nela, diferente do IF, não faz um comparativo pra saber se aquela expressão é true ou false, mas sim a expressão irá decidir qual ação será tomada. Abaixo terá exemplo de como essa estrutura funciona.

                        expressao(i)
                            |
                            |
                            |
                 |--------- + ---------|
                 |          |          |
                 |          |          |
              caso(1)     caso(2)    caso(3)

Vimos que a expressão acima tem o valor i, imaginamos que esse i tenha o valor 1 nele, seguindo essa estrutura, nessa situação o caso(1) que vai ser o ativo, pois a expressão tem o valor de 1. A estrutura Switch case trabalha assim, nela definiremos um valor para a expressão e esse valor ira definir qual caso será executado. LEMBRE-SE no final de cada caso utilize o comando "break", pois com ele o programa irá encerrar, se o break nao for utilizado o programa continuará rodando.
